# 合伙人 Review 指南

## 怎么打开

解压后，直接双击打开：

```text
index.html
```

如果浏览器限制本地文件，也可以在项目目录启动本地服务：

```bash
python3 -m http.server 8088
```

然后访问：

```text
http://localhost:8088
```

## 重点看什么

- `测评`：用户是否愿意填写，问题是否太多，商品图是否足够吸引。
- `成本表`：SKU、采购价、物流摊销、服务费率、损耗预留是否能支撑真实报价。
- `线索`：提交测评后，是否能沉淀微信、预算、家庭结构、预估毛利。
- `内容库`：小红书/抖音选题是否适合大同本地线上获客。
- `PLAN.md`：完整业务方案、每日小红书运营、30 天验证指标。

## Review 建议

请重点反馈三件事：

1. 这个定位和普通山姆代购相比，差异化是否足够清楚。
2. 用户看到商品和测评后，是否有动力留下微信。
3. 成本表里的价格、毛利率、履约成本逻辑是否能真实跑通。
