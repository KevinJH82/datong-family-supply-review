# 大同家庭补货助手落地方案

## 项目定位

面向大同中高消费家庭，做重线上、轻线下的家庭补货清单服务。项目不定位为普通“山姆代购”，而是用内容平台承接用户，通过测评、小程序式页面、成本表和私聊确认完成成交。

核心链路：

```text
小红书/抖音内容 -> H5/小程序测评 -> 成本表试算 -> 私聊确认 -> 预售采购 -> 复购提醒
```

## 当前产品

- 用户端：家庭补货测评、商品橱窗、预算选择、推荐结果。
- 成本端：采购城市、物流摊销、服务费率、损耗预留、冷链包装费、SKU 当天采购价。
- 运营端：线索后台、状态管理、CSV 导出、内容素材库。
- 价格口径：`399/699/999` 是预算档，不是固定套餐；正式报价前必须替换成实际采购价。

## 落地流程

1. 每次采购前一天更新 SKU 成本表。
2. 用户从小红书/抖音进入测评页，填写家庭人数、孩子年龄、预算、品类、冷链需求、微信。
3. 系统按当前成本表生成试算包，显示商品成本、总成本、预估毛利和毛利率。
4. 私聊只确认预算、替换规则、到货时间和收货区域，不承诺固定 SKU。
5. 毛利率低于 12% 的订单不成交，改为提高预算、减少冷链或调整 SKU。

## 小红书每日运营

每日运营包包括：

- 当日选题：围绕 `大同山姆`、`大同家庭囤货`、`大同宝妈零食`、`大同有娃家庭补货` 等关键词。
- 标题：3-5 个可选标题。
- 正文：小红书口吻，偏清单、避坑、家庭场景，不硬广。
- 图片脚本：每张图拍什么、图上写什么字。
- 评论区引导：引导用户私信或进入测评页。
- 私信回复：覆盖价格、采购方式、冷链、是否官方、缺货替换等问题。
- 数据复盘：根据曝光、点赞、收藏、评论、私信、测评数、下单数调整次日内容。

建议周节奏：

- 周一：本周补货清单
- 周二：山姆商品避坑
- 周三：有娃家庭零食/早餐
- 周四：预算包案例
- 周五：周末到货预告
- 周六：真实到货/分拣图
- 周日：一周复盘/下周预告

每天给运营助手的信息：

```text
1. 今天想推哪些商品/预算包
2. 是否有真实图片或到货图
3. 昨天数据：曝光、点赞、收藏、评论、私信、测评数、下单数
```

## 30 天验证指标

- 测评提交：100 份
- 首单转化：20-30 单
- 复购或会员意向：5 个
- 测评到首单转化率：15% 以上
- 首单客单价：399 元以上
- 毛利率底线：12% 以上

## 合规边界

- 不宣称山姆官方授权。
- 不使用“官方价”“官方代购”等误导表达。
- 山姆只是供应链和内容关键词之一，不是唯一卖点。
- 涉及食品销售前，确认营业执照、食品经营许可或仅销售预包装食品备案要求。
- 保留采购小票、到货照片和售后记录。
