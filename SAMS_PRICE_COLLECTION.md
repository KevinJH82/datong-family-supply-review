# 山姆采价与毛利预测操作说明

## 结论

山姆中国官网公开 Web 页面没有稳定开放商品搜索、畅销排序和价格数据。山姆 App/小程序价格通常依赖城市、门店、会员、库存和优惠状态，不能用第三方聚合价替代真实采购价。

因此当前采用半自动方式：

```text
山姆 App/小程序人工采价
-> 填入 CSV
-> H5 成本表导入 SKU
-> 自动生成 399/699/999 元起预算包毛利预测
```

## 采价城市

按实际采购地采价，不要混用城市价格。

建议优先：

- 北京
- 天津
- 后续如果太原/周边有稳定采购点，再单独建一版价格表

## 采价品类

第一批采 30-50 个 SKU，不追求全量。

### 复购底盘品

- 零食：坚果、薯片、饼干、肉脯、海苔、儿童零食
- 饮品：牛奶、酸奶、果汁、咖啡、气泡水
- 早餐：面包、麦片、谷物、蛋白棒、早餐组合
- 纸品：抽纸、卷纸、厨房纸
- 清洁日用：洗衣液、洗洁精、湿巾、垃圾袋

### 内容引流品

- 瑞士卷
- 麻薯
- 烤鸡
- 牛肉卷
- 榴莲千层/蛋糕类

### 谨慎采价品

- 冷冻生鲜
- 冷藏乳品
- 短保烘焙
- 熟食

这类商品可以带来内容流量，但要单独标记风险，不能作为毛利主力。

## 如何判断“畅销”

如果山姆 App/小程序有官方排序，优先使用：

- 热销
- 销量
- 推荐
- 复购榜
- 本周爆品

如果没有明确排序，用以下信号近似：

- 搜索关键词靠前展示
- 首页/频道页露出
- 小红书/抖音反复出现
- 本地代购经常发
- 用户主动询问

注意：这些是运营判断，不等同于山姆官方畅销榜。

## CSV 字段

文件字段固定为：

```text
enabled,name,category,price,risk,note
```

字段解释：

- `enabled`：是否启用，填 `true` 或 `false`
- `name`：商品名
- `category`：早餐、儿童零食、饮品、纸品、清洁日用、宠物用品、单位福利
- `price`：山姆 App/小程序当天实付价
- `risk`：normal、cold、frozen、hot
- `note`：采价城市、规格、是否短保、是否引流品

风险标记：

- `normal`：常温/耐储
- `cold`：冷藏
- `frozen`：冷冻
- `hot`：熟食/时效敏感

## 毛利预测公式

```text
订单毛利 = 客户预算 - 商品采购成本 - 物流摊销 - 冷链包装费 - 损耗预留
订单毛利率 = 订单毛利 / 客户预算
```

判断线：

- 低于 10%：不建议成交
- 10%-12%：只适合作为获客试单
- 12%-18%：可持续观察
- 18% 以上：较好，但要复核是否低估了履约/售后

## 每次采购前流程

1. 打开山姆 App/小程序，切到实际采购城市/门店。
2. 按品类采 30-50 个 SKU。
3. 填入 `data/sams_price_template.csv`。
4. 打开 H5 的“成本表”，导入 CSV。
5. 更新物流摊销、服务费率、损耗预留、冷链包装费。
6. 用 399/699/999 元起预算包试算毛利。
7. 毛利率低于 12% 的组合不要对外推。
